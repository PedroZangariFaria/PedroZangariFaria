## 📌 Projetos em destaque

### 🍽️ Sistema de Gerenciamento de Restaurantes (Python + POO)

Projeto desenvolvido em **Python** com foco em **Programação Orientada a Objetos**.  
O sistema permite cadastrar restaurantes, gerenciar informações e organizar funcionalidades utilizando classes, métodos e estruturas de controle.

Principais conceitos aplicados:

- Programação Orientada a Objetos (classes, métodos e encapsulamento)
- Estruturas condicionais e de repetição
- Organização e modularização de código
- Simulação de um sistema real de gerenciamento

### 🧾 Sistema de Restaurantes – Lógica e Estrutura de Programação

Aplicação em **Python** desenvolvida para praticar fundamentos da programação e construção de sistemas no terminal.

O projeto utiliza:

- Funções
- Listas
- Estruturas condicionais
- Estruturas de repetição
- Organização de fluxo de execução

O objetivo foi consolidar **lógica de programação e estruturação de sistemas simples**, base essencial para projetos mais complexos.
